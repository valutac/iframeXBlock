# iFrame XBlock

## Installation

Clone the project.

```bash
$ git clone https://github.com/valutac/iframeXBlock.git
```

Install XBlock

```bash
$ sudo /edx/bin/pip.edxapp install -e ./iframeXBlock
```

## Activate the XBlock in course
Go to Settings -> Advanced Settings and set advanced_modules to `["edx_iframe"]`.
